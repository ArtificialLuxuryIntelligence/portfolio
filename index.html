Vocabify

Description

A web application designed to help language users learn new vocabulary based on the current level of their target language. Currently available in English, French and Spanish.

Users can submit text and will be returned a list of words with their definitions from the text that they, most likely, do not know.

New vocabulary is proposed to the user based on their language level.

Technical

  • MERN stack
  • MongoDB Atlas
  • JSON Web Tokens
  • bcryptjs
  • Fully responsive design
  • Server-side code determines what vocabulary to send to user based on input text, user data and word frequency lists.
  • Backend uses (Unofficial) Google dictionary API

Squiggler

Description

A drawing game where visitors can complete drawing a doodle based on a 'squiggle' created by another user.

A separate gallery section shows some of the creations.

Users can log in, create private games and invite friends to play.

note: best experience on mobile

Technical

  • Express with Handlebars templating
  • MongoDB Atlas
  • Passport.js authentication
  • Fully responsive design
  • Canvas API with custom drawing (with smoothing)
  • Infinite scroll gallery
